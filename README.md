# AI System Frontend

A React-based frontend application for an AI proposal system with history tracking.

## Features

- **Proposal Page**: Main interface for AI proposal generation
- **History Page**: Sidebar to view and manage proposal history
- **PDF Export**: Generate PDF documents from proposals
- **Responsive Design**: Built with Tailwind CSS

## Tech Stack

- **React** 19.2.0
- **Vite** 7.3.1 - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Axios** - HTTP client
- **jsPDF & html2canvas** - PDF generation

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```
